1. Introduction to MS Access

Microsoft Access is a database management system (DBMS) that combines the relational Microsoft Jet Database Engine with a graphical user interface and software-development tools. It is used to store, manage, and analyze data.

  • Part of Microsoft Office Suite
  • Used to create and manage databases
  • Provides tools for creating tables, queries, forms, and reports
  • Supports data storage, retrieval, and manipulation
  • Suitable for small to medium-sized database applications

2. MS Access Interface

Important components of MS Access interface:

  • Ribbon: Contains tabs like File, Home, Create, External Data, Database Tools
  • Navigation Pane: Lists all database objects (tables, queries, forms, reports)
  • Document Tabs: Show open database objects for easy switching
  • Property Sheet: Displays properties for selected objects
  • Status Bar: Information about current mode and progress

3. Database Basics

  • Database: Collection of related data organized in tables
  • Table: Stores data in rows (records) and columns (fields)
  • Field: Single piece of data or attribute e.g., Name, Age
  • Record: Complete set of fields (a row)
  • Primary Key: Unique identifier for records in a table
  • Relationships: Connections between tables to maintain data integrity

4. Creating and Managing Tables

  • Creating tables in Design View or Datasheet View
  • Defining field names and data types (Text, Number, Date/Time, Currency, etc.)
  • Setting primary key
  • Entering and editing data in tables
  • Using Lookup fields to link data
  • Indexing fields to improve performance

5. Queries

  • Create queries to retrieve specific data or perform actions
  • Select Query: Retrieve data based on criteria
  • Action Queries: Make changes such as Update, Delete, Append, Make-Table
  • Using criteria and operators to filter data
  • Using expressions to calculate values
  • Sorting results in queries

6. Forms

  • Forms provide user-friendly interface to input and view data
  • Creating forms using Form Wizard or Design View
  • Adding controls like text boxes, combo boxes, buttons
  • Customizing layout and design
  • Using forms for data entry and navigation

7. Reports

  • Create reports to format, summarize, and print data
  • Using Report Wizard or Design View
  • Grouping and sorting data in reports
  • Adding headers, footers, and calculated controls
  • Exporting reports to PDF or other formats

8. Relationships and Referential Integrity

  • Creating relationships between tables using primary and foreign keys
  • Enforcing referential integrity to maintain consistency
  • Cascade update and delete options

9. Macros and Modules

  • Automate tasks with macros (no programming needed)
  • Creating, editing, and running macros
  • Advanced automation using VBA modules

10. Importing and Exporting Data

  • Import data from Excel, Text files, other databases
  • Export data and reports to Excel, PDF, Text, and other formats
  • Linking to external data sources

11. Security and Database Management

  • Setting passwords on database files
  • Backing up databases
  • Compacting and repairing database for performance and integrity
  • User-level security options

12. MS Access Keyboard Shortcuts

  • Ctrl + N: New database
  • Ctrl + O: Open database
  • Ctrl + S: Save object
  • Ctrl + P: Print
  • Ctrl + F: Find
  • Ctrl + H: Replace
  • Ctrl + C: Copy
  • Ctrl + X: Cut
  • Ctrl + V: Paste
  • Ctrl + Z: Undo
  • Ctrl + Y: Redo
  • Alt + Enter: Edit field in table
  • F11: Show or hide Navigation Pane
