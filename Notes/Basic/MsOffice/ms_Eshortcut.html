MS Excel Shortcut Keys with Uses

S.No. Shortcut Key Use
1Ctrl + NCreate a new workbook.
2Ctrl + OOpen an existing workbook.
3Ctrl + SSave the current workbook.
4F12Save As dialog box.
5Ctrl + PPrint the active worksheet.
6Ctrl + WClose the current workbook window.
7Ctrl + F4Close the active workbook window.
8Alt + F4Close Excel.
9Ctrl + CCopy selected cells.
10Ctrl + XCut selected cells.
11Ctrl + VPaste copied or cut cells.
12Ctrl + Alt + VOpen Paste Special dialog.
13Ctrl + ZUndo last action.
14Ctrl + YRedo last action.
15Ctrl + FFind dialog.
16Ctrl + HReplace dialog.
17Ctrl + ASelect all cells in worksheet.
18Ctrl + Shift + Arrow KeySelect to edge of data region.
19Ctrl + Arrow KeyMove to edge of data region.
20Ctrl + HomeGo to beginning of worksheet.
21Ctrl + EndGo to last cell with content.
22Page UpMove one screen up.
23Page DownMove one screen down.
24Alt + Page UpMove one screen left.
25Alt + Page DownMove one screen right.
26Ctrl + Page UpMove to previous worksheet tab.
27Ctrl + Page DownMove to next worksheet tab.
28Ctrl + Shift + Page UpSelect previous worksheet tab.
29Ctrl + Shift + Page DownSelect next worksheet tab.
30Ctrl + TabSwitch to next workbook window.
31Ctrl + Shift + TabSwitch to previous workbook window.
32Ctrl + BBold selected text.
33Ctrl + IItalic selected text.
34Ctrl + UUnderline selected text.
35Ctrl + 5Apply or remove strikethrough formatting.
36Ctrl + Shift + $Apply currency format.
37Ctrl + Shift + %Apply percent format.
38Ctrl + Shift + #Apply date format.
39Ctrl + Shift + @Apply time format.
40Ctrl + Shift + ^Apply exponential format.
41Ctrl + Shift + !Apply number format with two decimals.
42Ctrl + Shift + ~Apply general number format.
43Ctrl + Shift + &Apply outline border.
44Ctrl + Shift + _Remove outline border.
45Ctrl + 9Hide selected rows.
46Ctrl + 0Hide selected columns.
47Ctrl + Shift + (Unhide rows.
48Ctrl + Shift + )Unhide columns.
49Ctrl + DFill down from the cell above.
50Ctrl + RFill right from the cell to the left.
