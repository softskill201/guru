1. Introduction to MS Excel

Microsoft Excel is a powerful spreadsheet application used for data analysis, calculation, charting, and reporting. It is widely used in offices, educational institutes, and businesses to manage and analyze numerical data effectively.

  • Part of Microsoft Office Suite
  • Spreadsheets are organized in Workbooks and Worksheets
  • Supports powerful formulas, functions, and charts
  • Used for accounting, data entry, statistical analysis, graphing, and more

2. Excel Interface and Ribbon

Familiarize yourself with the Excel components to work efficiently:

  • Title Bar: Displays workbook name and program info
  • Ribbon: Contains tabs like Home, Insert, Page Layout, Formulas, Data, Review, View
  • Quick Access Toolbar: Customize frequently used commands
  • Formula Bar: Shows and edits content of the active cell
  • Status Bar: Displays info like sum, average of selected cells
  • Worksheet Area: Cells organized in rows (numbers) and columns (letters)

3. Worksheets and Workbooks

  • Workbook: Excel file containing one or more worksheets
  • Worksheet: Individual sheets/tabs within a workbook
  • You can add, rename, move, or delete worksheets
  • Default worksheet count can vary; usually 1-3 sheets
  • Use sheet tabs at the bottom to navigate

4. Data Entry and Cell Basics

  • Click on a cell to make it active (borders highlight it)
  • Enter text, numbers, or formulas in cell
  • Press Enter to go down, Tab to move right
  • Cell references: e.g., A1, B2 refer to column and row
  • Use arrow keys or mouse to navigate cells

5. Formatting Cells

  • Change font style, size, color
  • Use bold, italic, underline
  • Adjust cell background color (Fill Color)
  • Number formats: General, Number, Currency, Date, Percentage etc.
  • Alignment: horizontal, vertical, wrap text, merge cells
  • Format Painter copies formatting from one cell to others

6. Formulas and Functions in MS Excel

Formulas in Excel always begin with an equal sign = and perform calculations using cell references, constants, operators, and functions.

6.1. Writing Basic Formulas

  • Start with = followed by expression or function
  • Example: =A1+B1 adds values in cells A1 and B1
  • Operators:
    • + Addition
    • - Subtraction
    • * Multiplication
    • / Division
    • ^ Exponentiation (power)
  • Use parentheses () to control calculation order
  • Example: =(A1+B1)*C1 adds A1 and B1, then multiplies the result by C1

6.2. Relative, Absolute, and Mixed References

  • Relative Reference: Changes when copying formulas (e.g., A1)
  • Absolute Reference: Stays fixed when copying formulas (e.g., $A$1)
  • Mixed Reference: Fixes either row or column (e.g., $A1 or A$1)
  • Use F4 key to toggle between references while editing formula

6.3. Common Functions and Their Use Cases

Function Syntax Description & Example
SUM =SUM(range) Adds all numbers in the specified range.
Example: =SUM(A1:A5) sums values from A1 to A5.
AVERAGE =AVERAGE(range) Calculates the average of numbers.
Example: =AVERAGE(B1:B10).
IF =IF(logical_test, value_if_true, value_if_false) Checks a condition and returns one value if TRUE, another if FALSE.
Example: =IF(A1>50, "Pass", "Fail") returns "Pass" if A1 is greater than 50.
VLOOKUP =VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup]) Looks up a value vertically in the first column of a range and returns a value in the same row from another column.
Example: =VLOOKUP(1001, A2:C10, 3, FALSE) looks 1001 in first column, returns column 3 value.
HLOOKUP =HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup]) Searches a value horizontally in the top row of table and returns value from specified row.
Example: =HLOOKUP("Jan", A1:D4, 3, FALSE).
CONCATENATE (or CONCAT) =CONCATENATE(text1, text2, ...) or =CONCAT(text1, text2, ...) Joins multiple text strings into one.
Example: =CONCATENATE(A1, " ", B1) joins the contents of A1 and B1 with a space.
LEN =LEN(text) Returns the length (number of characters) of text.
Example: =LEN(A1).
TODAY =TODAY() Returns the current date.
Updates automatically each day.
NOW =NOW() Returns current date and time.
Automatically updates.
ROUND =ROUND(number, num_digits) Rounds a number to specified digits.
Example: =ROUND(A1, 2) rounds to 2 decimals.
COUNT =COUNT(range) Counts number of numeric entries in a range.
Ignore text and blanks.
COUNTA =COUNTA(range) Counts all non-empty cells (numbers, text, etc.) in a range.

6.4. Logical & Comparison Operators

  • = Equal to
  • <> Not equal to
  • > Greater than
  • < Less than
  • >= Greater than or equal to
  • <= Less than or equal to

6.5. Text Functions

  • =UPPER(text) - converts text to uppercase.
  • =LOWER(text) - converts text to lowercase.
  • =PROPER(text) - capitalizes the first letter of each word.
  • =LEFT(text, num_chars) - returns first characters.
  • =RIGHT(text, num_chars) - returns last characters.
  • =TRIM(text) - removes extra spaces.

6.6. Date & Time Functions

  • =DATE(year, month, day) - constructs a date from parts.
  • =YEAR(date), =MONTH(date), =DAY(date) - extract parts from date.
  • =NETWORKDAYS(start_date, end_date) - counts working days between dates.
  • =NOW() and =TODAY() - current date and time.

6.7. Example Formula Use Cases

  • Calculate total sales for a range:
    =SUM(B2:B15)
  • Find average marks of students:
    =AVERAGE(C2:C20)
  • Determine eligibility:
    =IF(D2>=60, "Pass", "Fail")
  • Look up price for product code:
    =VLOOKUP(E2, ProductTable, 3, FALSE)
  • Concatenate first and last names:
    =CONCATENATE(A2, " ", B2)
  • Extract year from date of birth:
    =YEAR(C2)

7. Working with Rows and Columns

  • Select entire row or column by clicking on row number or column letter
  • Insert or delete rows and columns
  • Resize rows and columns by dragging boundary lines
  • Hide or unhide rows/columns
  • Freeze panes to keep rows or columns visible while scrolling

8. Charts and Graphs

  • Insert various chart types: Column, Line, Pie, Bar, Area, Scatter etc.
  • Select data range and choose Insert > Chart
  • Customize chart titles, labels, and styles
  • Move and resize charts on worksheet

9. Sorting and Filtering Data

  • Sort data ascending or descending based on selected columns
  • Filter data to view rows meeting criteria
  • Use AutoFilter on the Data tab
  • Clear filter to display all data

10. Printing and Page Setup

  • Setup margins, orientation, page size from Page Layout tab
  • Set print area to print selected cells
  • Preview page layout and breakpoints before printing
  • Add headers and footers
  • Fit content to one page width or height

11. Basic Data Analysis Tools

  • AutoSum: Quickly add a column or row of numbers
  • Conditional Formatting: Highlight data based on rules
  • Data Validation: Create dropdown lists and restrict data entry
  • Freeze Panes: Keep headings visible while scrolling
