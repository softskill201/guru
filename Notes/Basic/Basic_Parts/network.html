What is a Computer Network?

A computer network is a collection of interconnected computers and devices that communicate and share resources (data, files, printers, internet) using communication channels (wired or wireless). Networks enable collaboration, resource sharing, and communication locally and globally.

Types of Networks

  • PAN (Personal Area Network): Very small, connects devices within a few meters (e.g., Bluetooth, USB, hotspot).
  • LAN (Local Area Network): Covers a small area like a room, building, or campus (e.g., school/office network, Wi-Fi).
  • MAN (Metropolitan Area Network): Spans a city or large campus (e.g., city-wide cable TV, university network).
  • WAN (Wide Area Network): Covers large areas, even countries/continents (e.g., Internet, bank networks).
  • Other: CAN (Campus Area), SAN (Storage Area), VPN (Virtual Private Network).

Network Topologies

  • Bus Topology: All devices connected to a single backbone cable.
  • Star Topology: All devices connected to a central hub/switch.
  • Ring Topology: Each device connected to two others, forming a ring.
  • Mesh Topology: Every device connects to every other device (robust, expensive).
  • Hybrid Topology: Combination of two or more topologies.

Network Devices

  • Hub: Simple device that connects multiple computers in a LAN (broadcasts data to all).
  • Switch: Smarter than hub, sends data only to intended device.
  • Router: Connects different networks, routes data between LANs, WANs, and the Internet.
  • Bridge: Connects and filters traffic between two LAN segments.
  • Repeater: Boosts signals to extend network range.
  • Modem: Converts digital data to analog for transmission over phone lines (and vice versa).
  • Access Point: Wireless device that allows Wi-Fi devices to connect to a wired network.
  • Firewall: Security device to monitor and control incoming/outgoing network traffic.

Transmission Media

  • Wired: Twisted pair cables (Ethernet), coaxial cables, fiber optic cables.
  • Wireless: Radio waves (Wi-Fi), microwaves, infrared, Bluetooth, satellite.
  • Fiber Optic: Uses light signals, very high speed, long distance, immune to EMI.

Network Protocols

  • TCP/IP: Foundation of the Internet, ensures reliable data transmission.
  • HTTP/HTTPS: Protocols for web browsing (secure version is HTTPS).
  • FTP: File Transfer Protocol, for uploading and downloading files.
  • SMTP/POP/IMAP: Email protocols (sending and receiving emails).
  • Ethernet: Standard for wired LANs.
  • Wi-Fi (IEEE 802.11): Standard for wireless LANs.
  • Bluetooth: Short-range wireless communication.

Advantages of Computer Networks

  • Resource sharing (printers, files, internet)
  • Centralized data management and backup
  • Communication (email, chat, video calls)
  • Cost-effective and efficient collaboration
  • Remote access and cloud computing

Disadvantages & Risks

  • Security threats (hacking, viruses, data theft)
  • Network congestion and downtime
  • Initial setup and maintenance cost
  • Data privacy issues
  • Dependence on network for daily tasks

Frequently Asked Questions (FAQs)

  • Q1. What is the Internet?
    Ans: The Internet is a global WAN connecting millions of networks and devices worldwide.
  • Q2. What is the difference between a hub and a switch?
    Ans: A hub broadcasts data to all devices; a switch sends data only to the intended device.
  • Q3. What is an IP address?
    Ans: An IP address is a unique identifier for a device on a network (e.g., 192.168.1.1).
  • Q4. What is a firewall?
    Ans: A firewall is a security device that monitors and controls network traffic.
  • Q5. What is bandwidth?
    Ans: Bandwidth is the maximum data transfer rate of a network, measured in Mbps or Gbps.
  • Q6. What is a VPN?
    Ans: VPN (Virtual Private Network) creates a secure, encrypted connection over the Internet.
